![image.png](https://ipfs.busy.org/ipfs/QmbtHbmsDGR6HxmW33J8KmmASrAYSY2xLfg6qqUeJYMkbr)
## - 각 기능들을 빠르게 작동시킬 수 있도록 "단축키"를 추가!! 했습니다.
## - Alt(Mac은 Options) + 1 => User Shortcut 실행
## - Alt(Mac은 Options) + 2 => Tag Shortcut 실행
## - Alt(Mac은 Options) + 3 => Tag Filter실행
## - Alt(Mac은 Options) + 4 => Get Markdown Text 실행
## - Alt(Mac은 Options) + 5 => Go Family Sites 실행
## - ESC                                      => 열린 창 닫기
- 이제 단축키를 이용해서 빠르게 기능을 작동 시킬 수 있습니다.(ESC로 쉽게 닫을 수도 있고요.)
## - 단축키 기능은 설정에서 아이콘 화면을 안보이게해도 작동합니다.(단축키 쓰실 분들은 그냥 기능 아이콘들은 안보이게 하시는게 더 좋겠지요!!)
